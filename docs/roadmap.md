# Project Roadmap

## Phase 1: Foundation (Completed ✅)
- Core CRM modules: Attendance, Visitors, Absentees.
- Authentication system with role-based access.
- Basic reporting and CSV exports.

## Phase 2: Professionalization (Completed ✅)
- White-label engine (Global Text Dictionary).
- Branding controls (Colors, Logo, Favicon).
- Enhanced Reports with Print-CSS and KPI ribbons.
- Advanced staff management and credential resets.

## Phase 3: Data Integration (Current ⏳)
- Transition from LocalStorage to a real-time Database (PostgreSQL/Supabase).
- Implement real SMTP for automated email reports.
- Multi-tenancy support for church branches.

## Phase 4: AI & Automation (Future 🔮)
- **Gemini Integration**: Automated summary of service notes.
- **Predictive Analytics**: Forecasting attendance based on historical patterns.
- **Voice-to-Data**: Usher voice input for headcount recording.
- Mobile App wrapper (Capacitor/React Native).
