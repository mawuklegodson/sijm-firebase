# Salvation In Jesus Ministry - Church Management System (CMS)

Salvation In Jesus Ministry is a high-performance, white-label administration and analytics platform designed for modern ministry operations. It empowers church leadership and volunteer staff with real-time data tracking, visitor management, and executive-level intelligence.

## 🚀 Key Features

- **Dynamic Executive Dashboard**: Real-time KPIs on attendance trends, visitor conversion, and system health.
- **White-Label Customization**: Administrators can customize branding colors, logos, favicons, and every text string across the portal via a "Global Dictionary".
- **Attendance Intelligence**: Service-by-service headcount tracking with offline-first support and automatic synchronization.
- **Visitor Retention (First-Timers)**: Automated follow-up queues and interaction logging to ensure no visitor is overlooked.
- **Infrastructure Management**: Asset tracking with valuation audits and condition monitoring.
- **Communications Hub**: Centralized request system for equipment, maintenance, and announcements.
- **Reporting Engine**: One-click executive summaries, CSV data exports, and print-ready audit logs.

## 🛠 Tech Stack

- **Frontend**: React (ESM)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Analytics**: Recharts
- **State Management**: Context-based custom store with LocalStorage persistence.

## 🔒 Security
- Role-based access control (Admin vs. Usher).
- Temporary credential provisioning system.
- Standard institutional security protocols for password resets.
